Use this checklist on a real iPhone running the current TestFlight build.

This is intended for cofounder-style product signoff, not deep engineering QA.

## How To Use This Checklist

- Mark anything that feels broken, confusing, ugly, slow, or surprisingly awkward.
- Treat "this technically works, but feels weird" as a valid bug.
- Add screenshots or screen recordings for anything that is hard to describe.
- If an item partially works, note exactly where the experience breaks down.

## 1. Install And First Launch

- [ ] Install from TestFlight without issues.
- [ ] App launches successfully from the Home Screen.
- [ ] First impression feels polished and intentional.
- [ ] Camera permission prompt appears when expected.
- [ ] Granting camera access enters the camera smoothly.
- [ ] If permission is denied, the app explains what to do next clearly.

## 2. Core Camera Feel

- [ ] Live preview appears quickly.
- [ ] Camera screen feels clean and not cluttered.
- [ ] UI feels closer to native iPhone Camera than a tool-heavy app.
- [ ] Black top and bottom bands feel correctly sized.
- [ ] Shutter button feels centered and easy to hit.
- [ ] Controls feel visually balanced.
- [ ] Nothing important feels too small or too hidden.

Notes:

## 3. Coach Mode

- [ ] Coach mode can be selected from the setup sheet.
- [ ] Guidance prompt at the top is readable and helpful.
- [ ] The silhouette appears when it should.
- [ ] The silhouette generally follows the real subject in frame.
- [ ] Guidance feels supportive, not bossy or robotic.
- [ ] The coach voice sounds pleasant and natural.
- [ ] The camera still feels primary, with guidance layered on top.

Notes:

## 4. Templates Mode

- [ ] Templates mode can be selected from the setup sheet.
- [ ] Template overlays render correctly.
- [ ] Switching templates updates the framing immediately.
- [ ] Templates feel useful rather than distracting.
- [ ] The template experience still feels like a camera app, not a design tool.

Notes:

## 5. Setup And Controls

- [ ] Setup sheet opens and closes reliably.
- [ ] Burst options are easy to understand.
- [ ] Voice guidance toggle works.
- [ ] Zoom control in setup works as expected.
- [ ] Hiding advanced controls from the main camera feels like the right choice.
- [ ] Nothing important feels impossible to discover.

Notes:

## 6. Capture Flow

- [ ] Single photo capture works.
- [ ] Burst capture works.
- [ ] Captures complete without freezing the app.
- [ ] The app feels responsive right after taking a photo.
- [ ] The transition into review/history feels smooth.

Notes:

## 7. Review, History, And Export

- [ ] Captured shots show up in history.
- [ ] Opening an older session works.
- [ ] Crop preview modes look correct.
- [ ] Export or save flow works.
- [ ] Exported photos look correct in Photos or the share target.

Notes:

## 8. Reliability Checks

- [ ] No crashes during normal use.
- [ ] No camera freezes or black preview.
- [ ] Backgrounding and reopening the app works.
- [ ] Rotating the phone does not break the layout.
- [ ] App still behaves well after several captures in a row.
