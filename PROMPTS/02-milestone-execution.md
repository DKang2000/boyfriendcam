# Milestone Execution Prompt

Implement exactly one approved BoyfriendCam milestone at a time.

Requirements:

- Stay within the approved milestone scope
- Preserve modular boundaries between config, analysis, guidance, capture, and UI
- Run lint/tests at the end of the milestone
- Summarize changed files, commands run, open risks, and what needs review
- Append `VIBECODE_LOG.md` with the prompt used, what changed, manual review items, and remaining risks
- Stop after the milestone and wait for review before continuing
